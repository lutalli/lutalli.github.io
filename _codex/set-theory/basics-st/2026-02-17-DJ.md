> {{ "DJ#DEF:Disjoint Sets" | cod }}
>
> {:.no-indent}
> - $X$ and $Y$ are **disjoint** if
>
>   $$ X\cap Y = \varnothing. $$
>
> - $X$ is a **disjoint collection** if
>
>   $$ \bigcap X = \varnothing. $$
>
> - Not disjoint sets are called **joint** / **overlapping**.

---

> {{ "DJ#PROP-SLF" | cod }}
>
> $\varnothing$ is the only set disjoint to itself.

**Proof.**{:.run-out}

- $$ \varnothing\cap\varnothing = \varnothing. $$

- If $X$ is disjoint to itself,

  $$ X = X\cap X = \varnothing. $$

&#8203;<qed />

> {{ "DJ#PROP-ALL" | cod }}
>
> $\varnothing$ is the only set disjoint to any other set.

**Proof.**{:.run-out}

- $\varnothing$ is disjoint to any set by [IT#PROP-EMP (II)](/codex/IT#PROP-EMP).

- If $X$ is non-empty,

  $$ Y = X\cup\{X\} $$

  is always joint to $X$. By [ZF#PROP-SLF](/codex/ZF#PROP-SLF), $X\neq Y$.

&#8203;<qed />
