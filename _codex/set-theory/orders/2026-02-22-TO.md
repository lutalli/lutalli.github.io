> {{ "DEF-TO:Total Order" | codexify }}
>
> {:.no-indent}
> - A **total order** / **linear order** is a strongly connected partial order.
>
> - A **strict total order** / **strict linear order** is a connected strict partial order.

---

> {{ "PROP-TO-S" | codexify }}
>
> $\preceq$ is a total order if and only if $\prec$ is a strict total order.
