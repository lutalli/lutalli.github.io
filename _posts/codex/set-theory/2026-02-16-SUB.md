> <span class="math-h">DEF-SUB. Subset ($\mathsf{ZF}$).</span>
>
> A set $X$ is a **subset** of a set $Y$, written $X\subseteq Y$, if
>
> $$ \forall x :\enspace x\in X \enspace\Rightarrow\enspace x\in Y. $$
> 
> $X$ is a **proper subset** of $Y$, written $X\subset Y$, if $X\subseteq Y$ and $X\neq Y$.

> <span class="math-h">PROP-SUB-SLF.</span>
>
> For any set $X$,
>
> $$ X\subseteq X. $$

> <span class="math-h">PROP-SUB-EMP.</span>
>
> $\varnothing$ is the only set that is a subset of any set. That is, for any set $X$,
>
> $$ X=\varnothing \enspace\Leftrightarrow\enspace \forall Y:X\subseteq Y. $$

> <span class="math-h">PROP-SUB-TR. Transitivity.</span>
>
> For any three sets $X$, $Y$ and $Z$,
>
> $$ X\subseteq Y\land Y\subseteq Z \enspace\Rightarrow\enspace X\subseteq Z. $$

> <span class="math-h">PROP-SUB-EXT.</span>
>
> For any two sets $X$ and $Y$,
>
> $$ X\subseteq Y\land Y\subseteq X \enspace\Leftrightarrow\enspace X=Y. $$
