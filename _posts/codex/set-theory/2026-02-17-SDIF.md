> <span class="math-h">DEF-SDIF. Set Difference ($\mathsf{ZF}$).</span>
>
> The **difference** $X-Y$, or $X\setminus Y$, is the set
>
> $$ \{x\in X:x\notin Y\}. $$

> <span class="math-h">PROP-SDIF-EMP.</span>
>
> For any set $X$:
>
> - $X-\varnothing=X$.
> - $\varnothing-X=\varnothing$.
> - $X-X=\varnothing$.

> <span class="math-h">PROP-SDIF-DJ.</span>
>
> If $X$ and $Y$ are disjoint sets, then $X-Y=X$ and $Y-X=Y$.

> <span class="math-h">PROP-SDIF-SUB.</span>
>
> If $X\subseteq Y$, then $X-Y=\varnothing$.
