> **DEF-SUR. Surjection.**{: .codex-h }
>
> A **surjection** / **onto function** is a [function](/codex/F) $f:X\to Y$ that is [surjective](/codex/RP#DEF-RP-T), i.e. for all $y\in Y$ there is an $x\in X$ such that $f(x)=y$. In other words, $\operatorname{ran}f=Y$.

> **REM-SUR.**{: .codex-h }
>
> Since the codomain is not an intrinsic property ([REM-F-COD](/codex/F#REM-F-COD)), not is surjectivity.
>
> We can make every function surjective by choosing its range as its codomain.
