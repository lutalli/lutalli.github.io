> **DEF-SUC. Successor Ordinal.**{: .codex-h }
>
> {: .no-indent }
> - The **successor** of an ordinal $\alpha$ is
>
>   $$ \operatorname{suc}\alpha = \alpha\cup\{\alpha\}. $$
>
> - $\alpha$ is a successor ordinal, if $\alpha=\operatorname{suc}\beta$ for some $\beta$.
>
> - $\alpha$ is a **limit ordinal** if it is not a successor ordinal.

---

> **PROP-SUC-EMP.**{: .codex-h }
>
> $0$ is a limit ordinal.

> **PROP-SUC.**{: .codex-h }
>
> Let $\alpha$ be an ordinal. The following statements are equivalent:
>
> {: .par-upper-roman }
> 1. $\alpha$ is a limit ordinal.
> 2. $\alpha=\sup\\{\beta:\beta<\alpha\\}$.
> 3. $\alpha=\bigcup\alpha$.
> 4. $\alpha$ has no <a href="/codex/MAX">maximum</a>.
