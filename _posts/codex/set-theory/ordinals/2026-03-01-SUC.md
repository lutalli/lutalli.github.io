> **DEF-SUC. Ordinal Successor.**{: .codex-h }
>
> {: .no-indent }
> - The **successor** of an [ordinal](/codex/ORD) $\alpha$ is
>
>   $$ \operatorname{suc}\alpha = \alpha\cup\{\alpha\}, $$
>
>   which is also an ordinal.
>
> - $\alpha$ is a **successor ordinal**, if $\alpha=\operatorname{suc}\beta$ for some ordinal $\beta$.
>
> - An ordinal $\alpha$ is a **limit ordinal** if it is not a successor ordinal ([LO](/codex/LO)).

---

> **PROP-SUC-INF.**{: .codex-h }
>
> For any ordinal $\alpha$,
>
> $$ \operatorname{suc}\alpha = \inf\{\beta:\beta>\alpha\}. $$
