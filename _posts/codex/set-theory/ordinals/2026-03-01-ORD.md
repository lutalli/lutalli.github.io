> {{ "DEF-ORD:Ordinal Number" | codexify }}
>
> {:.no-indent}
> - An **ordinal number** / **ordinal** is a set that is [transitive](/codex/TRA) and [strictly well-ordered](/codex/WO) by $\in$.
>
> - The [class](/codex/CLS) of all ordinals is denoted by $\mathbf{Ord}$.

> {{ "DEF-ORD-LT" | codexify }}
>
> For two ordinals $\alpha$ and $\beta$, we define
>
> $$ \alpha < \beta \enspace\leftrightarrow\enspace \alpha\in\beta. $$

---

> {{ "PROP-ORD-EMP" | codexify }}
>
> $0:=\varnothing$ is an ordinal.

> {{ "PROP-ORD-BF:Burali-Forti Paradox" | codexify }}
>
> $\mathbf{Ord}$ is a [proper class](/codex/CLS).

---

> {{ "PROP-ORD-EL" | codexify }}
>
> Every element of an ordinal is an ordinal.

> {{ "PROP-ORD-SUB" | codexify }}
>
> Let $\alpha$ and $\beta$ be ordinals.
>
> {:.par-upper-roman}
> 1. $\alpha\subseteq\beta$ or $\beta\subseteq\alpha$.
> 2. If $\alpha\subset\beta$, then $\alpha\in\beta$.

> {{ "PROP-ORD-TO" | codexify }}
>
> $<$ is a [total order](/codex/TO) on $\mathbf{Ord}$.

> {{ "PROP-ORD-S" | codexify }}
>
> For any ordinal $\alpha$,
>
> $$ \alpha = \{\beta:\beta < \alpha\}. $$

> {{ "PROP-ORD-IT" | codexify }}
>
> If $C$ is a non-empty [class](/codex/CLS) of ordinals, then:
>
> {:.par-upper-roman}
> 1. $\bigcap C$ is an ordinal.
> 2. $\bigcap C\in C$.
> 3. $\bigcap C=\inf C$.

> {{ "PROP-ORD-U" | codexify }}
>
> If $X$ is a non-empty set of ordinals, then:
>
> {:.par-upper-roman}
> 1. $\bigcup X$ is an ordinal.
> 2. $\bigcup X=\sup X$.
