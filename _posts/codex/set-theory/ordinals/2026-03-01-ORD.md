> **DEF-ORD. Ordinal Numbers.**{: .codex-h }
>
> An **ordinal number** / **ordinal** is a [transitive set](/codex/TRA) that is [strictly well-ordered](/codex/WO) by $\in$.

> **DEF-ORD-LT.**{: .codex-h }
>
> For two ordinals $\alpha$ and $\beta$, we define
>
> $$ \alpha < \beta \enspace\Leftrightarrow\enspace \alpha\in\beta. $$

---

> **PROP-ORD-EMP.**{: .codex-h }
>
> $0:=\varnothing$ is an ordinal.

> **PROP-ORD-EL.**{: .codex-h }
>
> Every element of an ordinal is an ordinal.

> **PROP-ORD-SUB.**{: .codex-h }
>
> Let $\alpha$ and $\beta$ be ordinals.
>
> - $\alpha\subseteq\beta$ or $\beta\subseteq\alpha$.
> - If $\alpha\subset\beta$, then $\alpha\in\beta$.

> **PROP-ORD-S.**{: .codex-h }
>
> For any ordinal $\alpha$,
>
> $$ \alpha = \{\beta:\beta < \alpha\}. $$

> **PROP-ORD-IT.**{: .codex-h }
>
> If $X$ is a non-empty set of ordinals, then:
>
> - $\bigcap X$ is an ordinal.
> - $\bigcap X\in X$.
> - $\bigcap X=\inf X$.

> **PROP-ORD-U.**{: .codex-h }
>
> If $X$ is a non-empty set of ordinals, then:
>
> - $\bigcup X$ is an ordinal.
> - $\bigcup X=\sup X$.

---

> **PROP-ORD-WO.**{: .codex-h }
>
> Every [well-ordered set](/codex/WO) is isomorphic to a unique ordinal number.
