> **DEF-EQU. Equinumerous Sets.**{: .codex-h }
>
> $X$ and $Y$ are **equinumerous**, written $X\sim Y$, if there exists a [bijection](/codex/BIJ) from $X$ onto $Y$.

> **DEF-EQU-IEQ.**{: .codex-h }
>
> We write $X\lesssim Y$ if there exists an [injection](/codex/INJ) from $X$ to $Y$.
