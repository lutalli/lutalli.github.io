> {{ "DEF-CARD:Cardinality" | codexify }} \[$\leftarrow\mathsf{AC}$\]
>
> If we assume $\mathsf{AC}$, then every set $X$ can be [well-ordered](/codex/WO) ([PROP-WOT](/codex/WOT)). By [PROP-OTP-WO](/codex/OTP#PROP-OTP-WO), there is at least one [ordinal](/codex/ORD) that is [equinumerous](/codex/EQU) to $X$. We define the **cardinality** of $X$ to be
>
> $$ \lvert X\rvert = \min\{\alpha\in\mathbf{Ord}:X\approx\alpha\}. $$

---

> {{ "PROP-CARD" | codexify }}
>
> For any two sets $X$ and $Y$:
>
> {:.par-upper-roman}
> 1. $X\approx Y$ if and only if $\lvert X\rvert=\lvert Y\rvert$.
> 2. $X\lesssim Y$ if and only if $\lvert X\rvert\leq\lvert Y\rvert$.
> 3. $X\lnsim Y$ if and only if $\lvert X\rvert<\lvert Y\rvert$.
