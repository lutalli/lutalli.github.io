> **DEF-AC-CF. Choice Function.**{:.codex-h}
>
> Let $X$ be a set of non-empty sets. A [function](/codex/F) is a **choice function** on $X$ if
>
> $$ \forall A\in X : f(A)\in A. $$

> **AX-AC. Axiom of Choice.**{:.codex-h}
>
> For any set of non-empty sets $X$, there exists a choice function on $X$.

Axiom of Choice is denoted by $\mathsf{AC}$. $\mathsf{ZF}$ together with $\mathsf{AC}$ is denoted by $\mathsf{ZFC}$.
