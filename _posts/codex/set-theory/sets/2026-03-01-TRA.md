> **DEF-TRA. Transitive Set.**{: .codex-h }
>
> A set $T$ is **transitive** if every element of $T$ is a subset of $T$.

---

> **PROP-TRA.**{: .codex-h }
>
> The following statements are equivalent:
>
> {: .par-upper-roman }
> 1. $T$ is transitive.
> 2. $\bigcup T\subseteq T$.
> 3. $T\subseteq\mathcal{P}(T)$.
