> **DEF-SUB. Subset.**{: .codex-h }
>
> {: .no-indent }
> - $X$ is a **subset** of $Y$, written $X\subseteq Y$, if
>
>   $$ \forall x :\enspace x\in X \enspace\Rightarrow\enspace x\in Y. $$
> 
> - $X$ is a **proper subset** of $Y$, written $X\subset Y$, if $X\subseteq Y$ and $X\neq Y$.

---

> **PROP-SUB-EMP.**{: .codex-h }
>
> $\varnothing$ is the only set that is a subset of all sets. That is, for any $X$,
>
> $$ X=\varnothing \enspace\Leftrightarrow\enspace \forall Y:X\subseteq Y. $$

---

> **PROP-SUB-REF. Reflexivity.**{: .codex-h }
>
> For any $X$,
>
> $$ X\subseteq X. $$

> **PROP-SUB-TR. Transitivity.**{: .codex-h }
>
> For any $X$, $Y$ and $Z$,
>
> $$ X\subseteq Y\land Y\subseteq Z \enspace\Rightarrow\enspace X\subseteq Z. $$

> **PROP-SUB-ATS. Antisymmetry.**{: .codex-h }
>
> For any $X$ and $Y$,
>
> $$ X\subseteq Y\land Y\subseteq X \enspace\Rightarrow\enspace X=Y. $$

As a result:

> **PROP-SUB-PO.**{: .codex-h }
>
> $\subseteq$ is a [partial order](/codex/PO) on $\mathbb{V}$.
