> <span class="math-h">DEF-RP.</span>
>
> Let $\sim$ be a [binary relation](/codex/R) on $X$. $\sim$ is
>
> - **reflexive**, if $x\sim x$ for all $x\in X$;
>
> - **irreflexive** / **strict**, if $x\not\sim x$ for all $x\in X$;
>
> - **symmetric**, if $x\sim y\Rightarrow y\sim x$ for all $x$, $y\in X$;
>
> - **antisymmetric**, if $(x\sim y\land y\sim x)\Rightarrow x=y$ for all $x$, $y\in X$;
>
> - **asymmetric**, if $x\sim y\Rightarrow y\not\sim x$ for all $x$, $y\in X$;
>
> - **transitive**, if $(x\sim y\land y\sim z)\Rightarrow x\sim z$ for all $x$, $y$, $z\in X$;
>
> - **connected**, if $x\neq y\Rightarrow(x\sim y\lor y\sim x)$ for all $x$, $y\in X$;
>
> - **strongly connected**, if $x\sim y\lor y\sim x$ for all $x$, $y\in X$;
>
> - **left-unique** / **one-to-many** / **injective**, if $(x\sim z\land y\sim z)\Rightarrow x=y$ for all $x$, $y$, $z\in X$;
>
> - **right-unique** / **many-to-one** / **functional** / **a [function](/codex/F)**, if $(z\sim x\land z\sim y)\Rightarrow x=y$ for all $x$, $y$, $z\in X$.

> <span class="math-h">DEF-RP-T.</span>
>
> Let $\sim$ be a relation on $X$ and $Y$. $\sim$ is
>
> - **left-total**, if
>
>   $$ \forall x\in X\,\exists y\in Y : x\sim y, $$
>
>   i.e. $\operatorname{dom}{\sim}=X$;
>
> - **right-total** / **onto** / **surjective**, if
>
>   $$ \forall y\in Y\,\exists x\in X : x\sim y, $$
>
>   i.e. $\operatorname{ran}{\sim}=Y$.

> <span class="math-h">REM-RP-T.</span>
>
> Notice that the left-total and right-total properties are not intrinsic, as they depend on the choices of $X$ and $Y$.
