> {{ "DEF-PO:Partial Order" | codexify }}
>
> {:.no-indent}
> - A **partial order** is an [antisymmetric](/codex/RCL) [preorder](/codex/PRO), i.e. a [binary relation](/codex/R) that is reflexive, transitive and antisymmetric.
>
> - A **strict partial order** is a binary relation that is irreflexive, transitive and asymmetric.

---

> {{ "PROP-PO-ASY" | codexify }}
>
> A relation is a strict partial order as soon as it is irreflexive and transitive.

**Proof.**{:.run-in}By [PROP-RCL-ITA](http://localhost:4000/codex/RCL#PROP-RCL-ITA).<qed />

> {{ "PROP-PO-SPO" | codexify }}
>
> $\preceq$ is a partial order if and only if $\prec$ is a strict partial order.
