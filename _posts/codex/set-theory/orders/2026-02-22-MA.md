> **DEF-MA. Maximal and Minimal Element.**{: .codex-h }
>
> Let $(X,\preceq)$ be a [preordered set](/codex/PRO) and $a\in X$.
>
> {: .no-indent }
> - $a$ is a **maximal element** of $X$ if
>
>   $$ \forall x\in X :\enspace a\preceq x \enspace\Rightarrow\enspace x\preceq a. $$
>
> - $a$ is a **minimal element** of $X$ if
>
>   $$ \forall x\in X :\enspace x\preceq a \enspace\Rightarrow\enspace a\preceq x. $$

---

> **PROP-MA-PO.**{: .codex-h }
>
> Let $(X,\preceq)$ be a [partially ordered set](/codex/PO) and $a\in X$. Because $\preceq$ is antisymmetric, we have:
>
> {: .no-indent }
> - $a$ is a maximal element of $X$ if and only if there is no other element $x\in X$ such that $a\preceq x$, i.e.
>
>   $$ \forall x\in X :\enspace a\preceq x \enspace\Rightarrow\enspace x=a. $$
>
> - $a$ is a minimal element of $X$ if and only if there is no other element $x\in X$ such that $x\preceq a$, i.e.
>
>   $$ \forall x\in X :\enspace x\preceq a \enspace\Rightarrow\enspace x=a. $$
