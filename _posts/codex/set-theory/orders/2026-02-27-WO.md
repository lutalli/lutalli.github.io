> **DEF-WO. Well-Order.**{: .codex-h }
>
> {: .no-indent }
> - A **well-order** on $W$ is a [total order](/codex/TO) such that every non-empty subset of $W$ has a least element.
>
> - $\prec$ is a **strict well-order** if $\preceq$ is a well-order.

---

> **PROP-WO-END.**{: .codex-h }
>
> If $(W,\preceq)$ is a well-ordered set and $f:W\to W$ is an [endomorphism](/codex/OMOR), then $x\preceq f(x)$ for every $x\in W$.

> **PROP-WO-AUT.**{: .codex-h } \[Corollary to [PROP-WO-END](/codex/WO#PROP-WO-END)\]
>
> The only [automorphism](/codex/OMOR) on a well-ordered set $W$ is the [identity function](/codex/ID) $\operatorname{id}_W$.

> **PROP-WO-ISO.**{: .codex-h }
>
> If two well-ordered sets $W$ and $W'$ are [isomorphic](/codex/OMOR), then the isomorphism is unique.

> **DEF-WO-SEG.**{: .codex-h }
>
> Let $(W,\preceq)$ be a well-ordered set and $u\in W$. The **initial segment** of $W$ given by $u$ is
>
> $$ \{x\in W : x\prec u\}. $$
>
> $S\subseteq W$ is an initial segment of $W$, if $S$ is the initial segment of $W$ given by $u$ for some $u$.

> **PROP-WO-SEG.**{: .codex-h }
>
> No well-ordered set is isomorphic to an initial segment of itself.

> **PROP-WO-C.**{: .codex-h }
>
> If $W$ and $W'$ are well-ordered sets, then exactly one of the following cases holds:
>
> {: .par-upper-roman }
> 1. $W$ is isomorphic to $W'$.
> 2. $W$ is isomorphic to an initial segment of $W'$.
> 3. $W'$ is isomorphic to an initial segment of $W$.
