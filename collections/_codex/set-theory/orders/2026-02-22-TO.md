> {{ "DEF-TO:Total Order" | codexify }}
>
> {:.no-indent}
> - A **total order** / **linear order** is a [partial order](/codex/PO) that is [strongly connected](/codex/RCL).
>
> - A **strict total order** / **strict linear order** is a [strict partial order](/codex/PO) that is [connected](/codex/RCL).

---

> {{ "PROP-TO-STO" | codexify }}
>
> $\preceq$ is a total order if and only if $\prec$ is a strict total order.
