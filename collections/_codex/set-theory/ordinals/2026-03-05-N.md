> {{ "DEF-N:Natural Numbers / Finite Ordinals" | codexify }}
>
> {:.no-indent}
> - We define
>
>   $$ \N = \bigcap\{X:\text{$X$ is inductive}\}. $$
>
>   [Axiom of Infinity](/codex/ZF#AX-ZF-INF) guarantees the existence of at least one [inductive set](/codex/IND).
>
> - $\omega:=\N$ is an [ordinal](/codex/ORD). When treating $\N$ as a set, the symbol $\N$ is used. When treating it as an ordinal, the symbol $\omega$ is used.
>
> - $n$ is a **natural number** / **finite ordinal** if $n\in\N$.
>
> - An ordinal is **infinite** if it is not finite.
>
> - We define
>
>   $$ 0=\varnothing, \quad 1=\suc0, \quad 2=\suc1, \quad 3=\suc2, $$
>
>   and so on.
>
> - We define
>
>   $$ \N^* = \N\setminus\{0\}. $$
