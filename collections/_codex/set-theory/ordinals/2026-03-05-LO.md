> {{ "DEF-LO:Limit Ordinal" | codexify }}
>
> An ordinal $\alpha$ is a **limit ordinal** if it is not $0$ nor a [successor ordinal](/codex/SUC), i.e. there is no $\beta$ such that $\alpha=\suc\beta$.

---

> {{ "PROP-LO" | codexify }}
>
> Let $\alpha>0$ be an ordinal. The following statements are equivalent:
>
> {:.par-upper-roman}
> 1. $\alpha$ is a limit ordinal.
> 2. $\alpha=\sup\\{\beta:\beta<\alpha\\}$.
> 3. $\alpha=\bigcup\alpha$.
> 4. $\alpha$ has no [maximum](/codex/MAX).
> 5. $\alpha$ is [inductive](/codex/IND).
