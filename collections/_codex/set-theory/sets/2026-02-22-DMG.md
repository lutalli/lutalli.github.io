> {{ "PROP-DMG:De Morgan's Laws (Set Theory)" | codexify }}
>
> For any $A$, $B\subseteq X$:
>
> {:.par-upper-roman}
> 1. $X\smallsetminus A\cup B = (X\smallsetminus A)\cap(X\smallsetminus B)$.
> 2. $X\smallsetminus A\cap B = (X\smallsetminus A)\cup(X\smallsetminus B)$.
>
> More generally, for any $A\subseteq\powerset(X)$:
>
> {:.par-upper-roman start="3"}
> 3. $X\smallsetminus \bigcup A = \bigcap_{Y\in A}(X\smallsetminus Y)$.
> 4. $X\smallsetminus \bigcap A = \bigcup_{Y\in A}(X\smallsetminus Y)$.
