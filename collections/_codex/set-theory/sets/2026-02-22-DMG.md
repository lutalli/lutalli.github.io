> {{ "PROP-DMG:De Morgan's Laws (Set Theory)" | codexify }}
>
> Let $X$ be a set and write
>
> $$ \overline{A} = X\setminus A. $$
>
> For any $A$, $B\subseteq X$:
>
> {:.par-upper-roman}
> 1. $\overline{A\cup B} = \overline{A}\cap\overline{B}$.
> 2. $\overline{A\cap B} = \overline{A}\cup\overline{B}$.
>
> More generally, for any $A\subseteq\powerset(X)$:
>
> {:.par-upper-roman start="3"}
> 3. $\displaystyle \overline{\bigcup A} = \bigcap_{Y\in A}\overline{Y}$.
>
> 4. $\displaystyle \overline{\bigcap A} = \bigcup_{Y\in A}\overline{Y}$.
