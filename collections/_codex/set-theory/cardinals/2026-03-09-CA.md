> {{ "DEF-CA:Cardinal Number" | codexify }}
>
> {:.no-indent}
> - An [ordinal](/codex/ORD) $\alpha$ is a **cardinal number** / **cardinal** if $\alpha\not\approx\beta$ for all $\beta<\alpha$.
>
> - Every [natural number](/codex/N) is a [cardinal](/codex/CA) and is called a **finite cardinal**.
>
> - A cardinal is **infinite** if it is not finite.

---

> {{ "PROP-CA-LO" | codexify }}
>
> Every infinite cardinal is a [limit ordinal](/codex/LO).

**Proof.**{:.run-in}By [PROP-SUC-CARD](/codex/SUC#PROP-SUC-CARD).<qed />

> {{ "PROP-CA-G" | codexify }}
>
> For every ordinal $\alpha$ there is a cardinal greater than $\alpha$.
