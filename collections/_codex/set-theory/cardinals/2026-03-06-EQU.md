> {{ "DEF-EQU:Equinumerosity" | codexify }}
>
> {:.no-indent}
> - $X$ and $Y$ are **equinumerous**, written $X\approx Y$, if there exists a [bijection](/codex/BIJ) from $X$ onto $Y$.
>
> - $X\lesssim Y$ if there exists an [injection](/codex/INJ) from $X$ to $Y$.
>
> - $X\lnsim Y$ if $X\lesssim Y$ and $X\not\approx Y$.

---

> {{ "PROP-EQU-SB:Schröder-Bernstein Theorem" | codexify }}
>
> For any $X$ and $Y$, if $X\lesssim Y$ and $Y\lesssim X$, then $X\approx Y$.

> {{ "PROP-EQU-SUB" | codexify }}
>
> If $X\subseteq Y$, then $X\lesssim Y$.

**Proof.**{:.run-in}$\id_X:X\to Y$ is an injection.<qed />
