> {{ "DEF-ALF:Aleph Number" | codexify }}
>
> {:.no-indent}
> - We define the **alpeh numbers** / **alephs** using [Transfinite Recursion](/codex/TREC):
>
>   {:.par-upper-roman}
>   1. $\aleph_0=\omega$;
>   2. $\aleph_{\suc\alpha}=\aleph_\alpha^+$;
>   3. $\aleph_\alpha=\sup\\{\aleph_\beta:\beta<\alpha\\}$ for every [limit ordinal](/codex/LO) $\alpha$.
>
> - We also define $\omega_\alpha=\aleph_\alpha$ for every ordinal $\alpha$. We use $\aleph_\alpha$ when referring to an aleph number, and $\omega_\alpha$ when referring to an order-type.

> {{ "PROP-ALF-WO" | codexify }}
>
> $X$ is [infinite](/codex/FIN) and [well-orderable](/codex/WO) if and only if $\lvert X\rvert$ is an aleph.

> {{ "PROP-ALF-CA" | codexify }} \[$\leftrightarrow\AC$\]
>
> Every [infinite cardinal](/codex/CA) is an aleph.
