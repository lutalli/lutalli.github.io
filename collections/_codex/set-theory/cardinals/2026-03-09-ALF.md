> {{ "DEF-ALF:Aleph Number" | codexify }}
>
> {:.no-indent}
> - We define the **alpeh numbers** / **alephs** using [Transfinite Recursion](/codex/TREC):
>
>   {:.par-upper-roman}
>   1. $\aleph_0=\omega$;
>   2. $\aleph_{\alpha+1}=\aleph_\alpha^+$;
>   3. $\aleph_\alpha=\sup\\{\aleph_\beta:\beta<\alpha\\}$ for every [limit ordinal](/codex/LO) $\alpha$.
>
> - We also write $\omega_\alpha=\aleph_\alpha$. We use $\aleph_\alpha$ when referring to an aleph number, $\omega_\alpha$ when referring to an ordinal.

> {{ "PROP-ALF-WO" | codexify }}
>
> Let $X$ be [infinite](/codex/FIN). $X$ is [well-orderable](/codex/WO) if and only if $\lvert X\rvert$ is an aleph.

> {{ "PROP-ALF-CA" | codexify }} \[$\leftrightarrow\AC$\]
>
> Every [infinite cardinal](/codex/CA) is an aleph.
